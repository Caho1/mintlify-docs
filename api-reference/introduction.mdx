## 欢迎使用 SuitMe API

SuitMe 提供了一套完整的 API 接口，用于数字形象管理、顾客管理和服装搭配等功能。

## 认证方式

所有 API 端点都使用 Bearer Token 进行身份验证。

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

## 主要功能

- **数字形象管理**：创建和编辑数字形象
- **顾客管理**：顾客信息的增删改查
- **服装搭配**：智能服装搭配推荐
- **任务管理**：异步任务状态查询
